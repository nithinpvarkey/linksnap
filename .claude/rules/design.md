# Design System

Never deviate from this design system without owner approval.

BRAND:
  Logo:           ⚡ LinkSnapr
  Tagline:        Paste any URL. Understand it instantly.
  Footer:         Powered by SnapKeep
  Watermark:      Powered by SnapKeep — on every free card always

COLOURS:
  Primary:        indigo-500
  Secondary:      purple-500
  Background:     white
  Primary text:   slate-900
  Secondary text: slate-500
  Tag background: indigo-100
  Tag text:       indigo-700
  Success:        green-500
  Error:          red-500
  Warning:        amber-500

TYPOGRAPHY:
  Font:           System font stack — no external fonts — faster loading
  Headings:       font-semibold
  Body:           font-normal
  Tags:           font-medium text-sm

DESIGN RULES:
  - Mobile first — always design for phone screen first then scale up
  - Clean minimal premium — must look like people would pay for it
  - No heavy animations — subtle transitions at 150 milliseconds maximum
  - Cards must look beautiful and feel worth sharing
  - Loading state always shown — never a blank screen under any circumstance
  - Use skeleton loaders not spinners
  - Error messages are friendly and human — never show technical details
  - Every component handles its own loading state and error state
  - Always design mobile breakpoint first then md then lg
